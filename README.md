Music Store SQL Data Analysis
Project Overview

This project analyzes a music store database using SQL to uncover insights about customer behavior, sales performance, and artist popularity. Queries were written to answer key business questions related to revenue, customer spending, and music genre trends.

Tools & Technologies

SQL

PostgreSQL

Data Analysis

Relational Database Queries

Business Questions Solved

Identify the senior-most employee based on job title.

Determine countries generating the highest number of invoices.

Find the top 3 highest invoice totals.

Identify the city generating the highest revenue.

Determine the highest spending customer.

Retrieve Rock music listeners and their contact details.

Identify top 10 artists producing the most Rock tracks.

Find songs longer than the average track length.

Project Outcome

The analysis demonstrates how SQL can be used to extract business insights from relational databases, helping organizations understand customer behavior, sales trends, and product popularity.
